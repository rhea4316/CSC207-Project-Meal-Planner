# Fonts Directory

This directory contains custom fonts for the Meal Planner application.

## Current Fonts

The following fonts are currently included:

### Poppins (for Headings)
- **Poppins-Medium.ttf** (Weight: 500)
- **Poppins-SemiBold.ttf** (Weight: 600)
- **Poppins-Bold.ttf** (Weight: 700)

### Inter (for Body Text & Numbers)
- **Inter-Regular.ttf** (Weight: 400)
- **Inter-Medium.ttf** (Weight: 500)
- **Inter-SemiBold.ttf** (Weight: 600)

## Font Usage

- **Poppins**: Used for headings, titles, buttons, and sidebar menu items
- **Inter**: Used for body text, numbers, and data display (optimal for readability)

## Fallback

If fonts are not found, the application will fall back to:
- System fonts: Segoe UI (Windows), San Francisco (macOS), Roboto (Linux)
- Sans-serif system default
