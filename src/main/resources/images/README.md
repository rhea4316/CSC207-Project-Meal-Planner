# Images and Icons

This directory contains image assets for the Meal Planner application.

## Structure

- `icons/` - Application icons and UI icons
- `placeholder-recipe.png` - Default image for recipes without photos

## Adding Images

When adding images:
1. Use PNG format for icons (supports transparency)
2. Use JPG for photos (smaller file size)
3. Keep file sizes reasonable (< 500KB)
4. Use descriptive filenames

## Icon Sizes

For best results, provide icons in these sizes:
- 16x16 - Small icons
- 32x32 - Standard icons
- 48x48 - Large icons
- 64x64 - Extra large icons

## Placeholder Images

Add placeholder images for:
- Recipe without image: `placeholder-recipe.png`
- User avatar: `placeholder-avatar.png` (optional)
